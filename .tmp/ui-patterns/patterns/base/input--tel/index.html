Basic telephone input type for a form. These can read for specific regex phone patterns, or by default, accept a numeric-only input.

Application Methods

  • .base--tel-input -

    Add this class to an element

  • .base--STYLED input[type="tel"] -

    <input type="tel"> tag inside a class of .base--STYLED

  • %base--tel-input -

    Extend this silent selector through Sass

Settings

  • input padding -

    Padding of input box

  • input border width -

    Border width of input box

  • input border color -

    Color of input box border

  • background color -

    Background color of input box
