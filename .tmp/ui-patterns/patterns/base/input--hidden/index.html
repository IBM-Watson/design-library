Application Methods

  • .base--hidden-input -

    Add this class to an element

  • .base--STYLED input[type="hidden"] -

    <input type="hidden"> tag inside a class of .base--STYLED

  • %base--hidden-input -

    Extend this silent selector through Sass

    Padding of input box

  • input border width -

    Border width of input box

  • input border color -

    Color of input box border

  • background color -

    Background color of input box
