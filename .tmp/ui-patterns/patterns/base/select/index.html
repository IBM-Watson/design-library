A flexible and configurable way of selecting an individual item from a list. Lists should be kept as short as possible.

Application Methods

  • .base--select -

    Add this class to an element

  • .base--STYLED select -

    <select> tag inside a class of .base--STYLED

  • %base--select -

    Extend this silent selector through Sass

Settings

  • input padding -

    Padding of input box

  • input border width -

    Border width of input box

  • input border color -

    Color of input box border and downward triangle

  • background color -

    Background color of input box
