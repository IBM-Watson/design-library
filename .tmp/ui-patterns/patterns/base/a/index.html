Tag to define a hyperlink

Link to something 
<a href="#" class="base--a">Link to something</a>

Anchor link, <a>, is the base styling for a link

Application Methods

  • .base--a -

    Add this class to an element

  • .base--STYLED a -

    <a> tag inside a class of .base--STYLED

  • %base--a -

    Extend this silent selector through Sass

  • `@include base--a([$color: null, $background: null]) -

    Include mixin to customize links. $color is text color and defaults to primary color setting, $background is background color link will be on and defaults to background color setting.

Settings

  • primary color -

    Link color

  • background color -

    Assists with creating illusion of cleared descenders for underline

  • bold weight font -

    Weight of link
