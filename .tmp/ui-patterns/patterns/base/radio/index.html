A form field option where only one choice may be selected

<input role="radio" class="base--radio" type="radio" id="rb1" name="rb" value="">
<label class="base--inline-label" for="rb1">Option 1</label>

A single radio button, <input type=["radio"]>, is an individual form field in which only one of the given options may be chosen. In order to allow efficient screen reader use, an aria-role of “radio” (or “radiomenuitem” if it is part of a menu selection) should be included: role="radio".

Application Methods

  • .base--radio -

    Add this class to an element

  • .base--STYLED input[type="radio"] -

    <input type="radio"> tag inside a class of .base--STYLED

  • %base--radio -

    Extend this silent selector through Sass

Settings

  • input border width -

    Color of radio border
