A radio group, role="radiogroup", contains a list of radio button elements, input[type="radio"], of which only one can be selected as the selected answer.

Application Methods

  • .radiogroup -

    Add this class to the element surrounding the group of radio buttons

  • .radiogroup--item -

    Add this class to each individual item housing a radio button

  • %radiogroup -

    Extend this silent selector through Sass

  • %radiogroup--item -

    Extend this silent selector through Sass
